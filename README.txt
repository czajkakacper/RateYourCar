PL
Aby uruchomić projekt RateYourCar należy:
1. Otworzyć Terminal, wejść w folder "server" oraz "client" poleceniem 'cd client' oraz 'cd server', następnie wprowadzić polecenie 'npm i'
2. następnie uruchomić poleceniem "npm start" zarówno folder server jak i client
3. W tle uruchomiona powinna być baza danych MongoDB (baza do projektu znajduje się pod nazwa "rate_your_car", podzielona jest na users, czyli zarejestrowanych użytkowników oraz opinions, czyli dodane opinie na stronie)

EN
To run RateYourCar project have to:
1. Open new terminal, enter the folder "server" and "client" command 'cd client' and 'cd server', next enter the command 'npm i'
2. Next run the project by command "npm start" both the server and client folders.
3. The database should be running in the background MongoDB (the database for the project is under the name "rate_your_car", it is divided into users and reviews folder)
